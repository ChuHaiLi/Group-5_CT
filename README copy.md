# Task 3 – Recommendation & Ranking

## Mục tiêu
- Xây dựng thuật toán xếp hạng và gợi ý địa điểm dựa trên profile người dùng.

## Công cụ
- Python 3.x
- Thư viện: json, math

## Phân rã module
1. Load data
2. User profile
3. Compute score
4. Filter & ranking
5. Explainability
6. Output

## Cách chạy
1. Đặt file `sample_places.json` trong folder `data/`
2. Chạy:
   python recommendation.py
3. Xem output top 3–5 địa điểm với explain tags
